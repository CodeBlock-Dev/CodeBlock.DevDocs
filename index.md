# Hello World Example in C#

This example demonstrates a basic "Hello World" program in C#.

## Code Example

```csharp
using System;

namespace HelloWorldApp
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
```
